### Roadmap overview

| Weeks | Phase | Deliverable |
| --- | --- | --- |
| 1–4 | Analysis & Design | BRD, SRS, SDD |
| 5–8 | Sprint Development | Alpha |
| 9–12 | Testing & UAT | Beta |
| 13–16 | Go-live | Production |

### Weekly milestones (lightweight)

- **Week 1:** Vision + problem statement, stakeholders, KPIs
- **Week 2:** BRD scope, out of scope, success metrics, approvals
- **Week 3:** SRS drafted and reviewed, MVP cut confirmed
- **Week 4:** SDD approved, definition of done agreed
- **Weeks 5–8:** Sprint 1–2 execution, Alpha release to internal users
- **Weeks 9–12:** System testing, UAT, Beta release
- **Weeks 13–16:** Go-live, monitoring, incident readiness, handover
