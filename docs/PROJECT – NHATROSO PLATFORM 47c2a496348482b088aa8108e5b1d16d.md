# PROJECT – NHATROSO PLATFORM

<aside>
🎯

**Purpose:** One workspace to run a 16-week product build end-to-end (BRD → SRS → SDD → Sprint → QA → Release → Handover) with enterprise clarity and startup simplicity.

</aside>

### Start here

- Use **ROADMAP 16 WEEKS** to align phase and deliverables.
- Capture strategy in **BRD & BUSINESS** (Week 1–2).
- Write requirements in **REQUIREMENTS (SRS)** (Week 3).
- Finalize solution in **ARCHITECTURE & SDD** (Week 4).
- Execute in **BACKLOG & SPRINT BOARD** (Week 5–16).
- Validate in **TEST & QA** (Week 8–12).
- Ship and operate in **RELEASE & OPERATION** (Week 13–16).
